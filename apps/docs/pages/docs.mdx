## Props

| Prop                   |       Type       |       Default       |             Description |
| :--------------------- | :--------------: | :-----------------: | ----------------------: |
| value                  |      string      | https://youtube.com | Text to generate the QR |
| size                   |      number      |         150         |          Size of the QR |
| ecLevel                |     ECLevel      |          M          |  Error correction level |
| enableCORS             |     boolean      |        false        |                         |
| quietZone              |      number      |          x          |                         |
| bgColor                |      string      |       #FFFFFF       |                         |
| fgColor                |      string      |       #000000       |                         |
| logoImage              |      string      |                     |                         |
| logoWidth              |      number      |          0          |                         |
| logoHeight             |      number      |          0          |                         |
| logoOpacity            |      number      |          0          |                         |
| logoOnLoad             |     function     |                     |                         |
| removeQrCodeBehindLogo |     boolean      |        false        |                         |
| logoPadding            |      number      |          0          |                         |
| logoPaddingStyle       | LogoPaddingStyle |         no          |                         |
| eyeRadius              |                  |         no          |                         |
| eyeColor               |                  |         no          |                         |
| qrStyle                |     QRStyle      |         no          |                QR style |

### Extra types

```tsc
type ECLevel = "L" | "M" | "Q" | "H";
type LogoPaddingStyle = "square" | "circle";
type QRStyle = "squares" | "dots";
```
